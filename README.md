# downloadTcVideo

> 一个可以下载腾讯视频的 node 程序

+ 此程序只做学习交流，🚫请勿商用。 视频版权为腾讯所有。
+ VIP 视频不能下载，这个不是本程序重点。请支持正版。

# Run

> yarn or npm  我习惯用 yarn 😜😝😜😝

1. yarn run install 安装依赖
2. yarn run start  运行程序
3. 输入你要下载的腾讯视频地址， 回车，即可下载。
